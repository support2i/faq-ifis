FAQ : E-learning

Quelle est la différence entre un test de positionnement et une évaluation ?

Il existe deux types de parcours : Avec et sans test de positionnement

Avec test de positionnement : vous commencez par un test (1 tentative possible).

  • Si vous obtenez 80 % ou plus, la formation est validée directement. Vous n’avez plus besoin de suivre le module ni de faire l’évaluation finale.
  • En cas d’échec du test, vous devrez suivre le module e-learning puis passer l’évaluation de connaissance (3 tentatives possibles).

    • Un test de positionnement est un raccourci pour valider une formation si vous l’avez déjà suivi et avec l’accord par votre chargé de formation.

    Pourquoi ma formation est-elle encore marquée « en cours » alors que je l’ai suivie ?

    Pas d’inquiétude : cela n’impacte pas la validation. Cela vous permet simplement de revenir sur la formation à tout moment.

    Ma synthèse affiche une croix rouge

    Cela signifie que vous avez déjà téléchargé votre synthèse au moins une fois. Vous pouvez la télécharger à nouveau autant de fois que nécessaire.

    Ma synthèse ne s’ouvre pas

    Il arrive que le téléchargement ne se lance pas automatiquement.

    • Cliquez sur le lien proposé : « cliquez sur ce lien afin de la télécharger ».
    • Sinon, suivez les instructions affichées pour y accéder manuellement.
    Ma vidéo ne charge pas

    Cela peut venir d’une connexion internet trop faible. Nos recommandations :

    • Évitez les connexions multiples sur le même réseau Wifi.
    • Rapprochez-vous de la borne Wifi si possible.
    • Privilégiez les horaires du matin pour suivre vos modules.
    • Utilisez Google Chrome, généralement plus stable pour nos contenus.
    Mon évaluation n’est pas disponible
    • Si vous avez un parcours de formation sans test de positionnement : L’évaluation reste grisée tant que vous n’avez pas lancé le module e-learning. Cliquez sur le module pour commencer.
    • Si vous avez un parcours de formation avec un test de positionnement : L’évaluation est grisée si vous avez validé le test de positionnement. La formation est donc considérée comme validée.
    J’ai été déconnecté·e pendant une évaluation

    Des déconnexions peuvent arriver et vous faire perdre des essais à vos positionnements et évaluations. Contactez le support informatique pour réactiver votre tentative : support-informatique@ifis.fr

    J’ai atteint le nombre maximum de tentatives

    Contactez le support pour demander une réouverture des tentatives : support-informatique@ifis.fr

    Comment savoir si j’ai validé toutes mes formations ?
    • Si vous avez un parcours sans test de positionnement : L’icône verte "validé" s’affiche à côté de l’évaluation. Votre score (≥ 80 %) figure sur votre attestation.
    • Si vous avez un parcours avec test de positionnement : Une icône verte "validé" s’affiche à côté du test de positionnement ou de l’évaluation. Le score validé apparaît aussi sur votre attestation de réussite.
    Je n’ai plus accès à mes formations

    La période de formation est probablement terminée. Contactez votre chargé(e) de formation pour plus d’informations.

    En cas de doute ou pour toute autre question, contactez-nous à l'adresse suivante : support-informatique@ifis.fr.
